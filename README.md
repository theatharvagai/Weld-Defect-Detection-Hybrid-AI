This project presents a novel **Hybrid AI Framework** for automated weld defect detection in industrial environments, integrating:

* Generative Adversarial Networks (GAN) for data balancing
* Convolutional Neural Networks (CNN) for feature extraction

The proposed system addresses class imbalance in defect datasets and significantly improves detection accuracy and robustness in real-world industrial inspection scenarios.

This hybrid architecture achieves superior performance in complex weld defect detection tasks.
